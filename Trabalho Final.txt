PROJETO FINAL DE FRONT-END
Criação de um site para uma
Clínica de Estética e Harmonização
Facial.

Requisitos: Usar as linguagens HTML, Css, JavaScript.

Crie um site para uma Clínica de Estética e Harmonização Facial, crie nome, logo, cores e estilos da sua preferência.
Segue um esboço dos itens mínimos para página web.
Os itens abaixo podem ser melhorados com mais opções à sua escolha.

Navi : Barra de Navegação com pelo menos 5 itens:

● Conheça nossa empresa: Home
● Cadastro
● Tratamentos para a Face
● Tratamentos para o Corpo
● Tratamentos Capilares
● Podologia
● Consulta Estética ou Nutrição

Cada item acima, quando selecionado, abre uma janela com 
itens e explicações dos diversos tipos e como a opção de ser selecionada. Exemplo:

Face

● Limpeza de Pele
● Tratamentos Anti-idade
● Botox
● Preenchimentos: Labial e Maxilares
● Fios de sustentação
● Sobrancelhas

Consulta:
O cliente entra com o dia, mês, ano e o horário que deseja. 
E também seleciona o médico que deseja consultar (criar uma listade médicos). 
Deverá conter também uma caixa de mensagem para o cliente escrever o motivo da consulta se achar necessário.

Cadastro:
● Nome:
● Endereço:
● Telefone:
● Data de Nascimento:

Main: Deverá conter 4 imagens e 4 textos, oferecendo uma promoção ou serviço para cada semana. Serão 4 modelos de promoções 
dentro de um carrossel. (BootStrap).

Section_1: Composta por 3 cards com figura e texto, com brinde por fidelidade. 
Ex:
1) Contrate um tratamento anti-idade e ganhe uma limpeza de pele facial.
2) Faça uma depilação completa e ganhe a cutilagem de pé e mão.
3) Compre um pacote de massagem corporal e ganhe um Banho de Lua.
4 - 

Section_2: Deve ter 3 links, que abre artigos da web sobre 
tratamentos como: lipoaspiração, harmonização facial, podologia....e tb 
com imagens.
Deve ser atraente para que o usuário sinta vontade de ler.
Deve conter um campo para o usuário comentar o artigo.
Torne essa seção agradável, estimulante. Um ponto de encontro
para a comunidade ligada a essa empresa.

Footer: Sua livre escolha, não esquecendo as redes sociais do estabelecimento. 
Felicitações para os aniversariantes do dia. Dia da mulher. Dia da Avó.....

Data de Entrega: Até o dia ...

Exigências:

● Para usar qualquer serviço da página, o usuário deverá primeiro se cadastrar.
● Ao finalizar a seleção de um serviço, deverá aparecer uma caixa de mensagem, com os itens selecionados para 
confirmação das escolhas. E após essa confirmação, outra caixa agradecendo o uso dos serviços.
● Códigos iguais invalidam todos os trabalhos, ficando ambos os alunos com zero.
● Aproveitem a oportunidade para desenvolver um excelente projeto como experiência para seu portifólio.
O site deve ser responsivo
O site deve estar no github
